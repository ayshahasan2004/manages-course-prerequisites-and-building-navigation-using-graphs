This is a C-based system managing courses and building navigation using graphs
It reads data from two input files containing buildings and courses
It builds an adjacency matrix to represent distances between buildings
It uses Dijkstra’s algorithm to compute the shortest path
It constructs a directed graph for course prerequisite relationships
It supports data loading, validation, and input checking
It verifies whether a building exists before processing
It implements topological sorting to order courses correctly
It determines a valid sequence of course completion
It uses a queue data structure for graph traversal operations
It builds graphs dynamically from parsed file input
It parses structured text using string tokenization techniques
It computes shortest paths using cost arrays
It uses parent arrays to reconstruct the shortest route
It prints the final path between two buildings
It maintains a global array for building storage
It stores distances in an adjacency matrix representation
It handles courses with no prerequisites first in sorting
It uses dynamic memory allocation for queues and graph nodes
It provides a menu-driven interface with error handling and validation
